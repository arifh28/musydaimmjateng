Iftitah

IMM Setengah Abad sebagai bagian dari IMM dan menisbahkan diri untuk dunia desain dan digital kreatif. Kami akan membagikan sebuah web aplikasi untuk kegiatan permusyaratan, dalam hal ini adalah Musyawarah Daerah.

Seperti yang kami sebutkan diatas untuk kegiatan permusyaratan Musyawarah Daerah. Kenapa harus Musyda bukan Musycab? Berikut alasannya:

  • Musyawarah Daerah yang pesertanya satu provinsi dengan jumlah lebih dari 100 orang dengan banyak Pimpinan Cabang.
  • Dengan teritorial DPD yang luas, kader bisa langsung mengetahui mengenai kegiatan bersamanya.

Musyawarah Cabang bisa juga dibuatkan website yang seperti ini, karena pimpinan cabang masih mempunyai delegasi dari komisariatnya seperti layaknya Musyawarah Daerah dengan delegasi dari Pimpinan Cabang.

Keuntungan

Immawan/Immawati, dengan menggunakan website sebagai basis informasi keuntungannya antara lain:

  • Panitia bisa langsung menghitung jumlah peserta dengan pasti sehingga penempatan ruang istirahat bisa diketahui.
  • Semua kader bisa menjangkaunya, karena berbasiskan website.
  • Panitia Pemilihan bisa menghitung jumlah calon formatur yang masuk setiap Pimpinan Cabang langsung dari Website.
  • Pimpinan Cabang dengan mudah mengupload berkas calon formatur dan daftar peserta Musyda.
  • Dilengkapi dengan Admin panel untuk manajemen user Pimpinan Cabang.

Lisensi

Web aplikasi ini lisensinya MIT. Admin tidak membuat dari nol aplikasi website ini, namun modifikasi dari sourcecode yang dibagikan gedelumbung.com untuk keperluan IMM. Admin sudah meminta izin untuk bisa disebarkan secara gratis.

Installasi

Dalam installasi seperti website pada umumnya langsung di copy pada folder website. Jika Immawan/Immawati sudah punya webserver sendiri di internet, silahkan upload menggunakan aplikasi ftp kesayangan Immawan/Immawati. Tapi jika masih di webserver lokal dengan menggunakan Apache, PhpMyAdmin (XAMPP/LAMP/WAMP)langsung ditaruh di folder tersebut.

Karena ini menggunakan framework PHP Codeigniter, bagi Immawan/Immawati yang belum terbiasa menggunakan silahkan ikuti langkahnya dibawah ini.

  • Folder Aplikasi

Susunan folder Aplikasi yang ditaruh pada webserver adalah:

          .../application/
          .../asset/
          .../captcha/
          .../system/
          .../index.php
          .../.htacces
  • Konfigurasi Database

Database web aplikasi ini sudah disediakan dalam bentuk .sql dengan nama musyda.sql yang ada di GitHub Admin. Untuk konfigurasi sendiri Immawan/Immawati bisa lihat pada file database.php dalam folder application/config/database.php seperti ini.

      $db['default']['hostname'] = 'localhost';

      $db['default']['username'] = 'root';

      $db['default']['password'] = '';

      $db['default']['database'] = 'musyda';

      $db['default']['dbdriver'] = 'mysql';

Admin mencontohkan pada server lokal, jika Immawan/Immawati menyewa hosting sendiri hubungi penyedia hosting untuk ['hostname'] namun biasanya tetap 'localhost', untuk username dan password ada pada CPanel dengan membuatnya secara manual.

  • Menentukan Base Site URL

Framework CodeIgniter untuk menentukan konfigurasi Base URL ada pada application/config/config.php. Base URL ini berfungsi untuk menentukan alamat utama dari website Immawan/Immawati.

          $config['base_url']	= 'http://'.$_SERVER['SERVER_NAME'].'/musydaimm/';

Pada contoh menggunakan .$_SERVER['SERVER_NAME']. karena pada contoh web aplikasi ini bisa dikunjungi oleh komputer lain pada server lokal yang sama lewat jaringan LAN. Namun jika Immawan/Immawati menyewa hosting, baris ini diisi dengan nama domain/ subdomain website ini. Atau dimana Immawan/Immawati menaruh foldernya contohnya http://webpermusyawaratanimm.hol.es/.

    Pimpinan Cabang

    • Menambahkan Database Pimpinan Cabang dan Ketua Umumnya.

    Aplikasi ini masih belum sempurna, karena untuk menambahkan Pimpinan Cabang harus masuk langsung lewat database di phpmysql. Silahkan Immawan/Immawati tambahkan nama, nama ketua, alamat email dsb. pada tabel "dlmbg_cabang_profil".

    Setelah Immawan/Immawati menambahkan database Pimpinan Cabang, selanjutnya masuk sebagai Admin lalu pada Operator tambahkan operator mulai dari Nama Ketua, alamat email dsb.

    • Download

    Bagian Download ini tidak ada yang spesial hanya berupa gambar yang jika dilewati mouse akan berubah warna. Tapi link yang dipasang memang berasal dari Dropbox.com, karena selain terbatasnya space hosting juga untuk arsip IMM juga sehingga kita mudah mencarinya.

    • Calon Formatur

    Pada persyaratan Calon Formatur ini sudah sesuai Ketentuan Pemilihan Pimpinan dan AD/ART dalam Tanfidz XVI IMM. Beberapa point formulirnya pun tidak ada yang perlu dirubah. Namun untuk berkas formatur hanya bisa upload file .zip, .rar, .7zip atau .tar.gz. dengan ukuran maksimal 2 MB.

    • Daftar Peserta

    Pendaftaran peserta dengan formulir yang terdiri dari Nama, Immawan/Immawati, Asal Komisariat, dan No. HP. Dengan ketentuan tambahan surat Mandat, foto peserta dan tentunya SWO dan SWP.

    Dari Admin

    Aplikasi website ini masih jauh dari sempurna dan bagus. Pada kenyataannya, berkas calon formatur harus diserahkan juga dengan hard copy dari Pimpinan Cabang yang bersangkutan. Dan kesadaran Pimpinan Cabang tidak semuanya sama, sehingga tidak semua Pimpinan Cabang mendaftarkan peserta secara online. Acara permusyawaratan bukan hanya sekedar acara memilih Calon Formatur saja, tapi lebih kepada mempererat tali silaturahim antar cabang/komisariat dan hasil-hasil sidang komisinya.

    Dan yang terpenting adalah jangan pernah ada IMM sampai memunculkan politik uang apalagi hanya tingkat DPD. Selain itu juga, Musyda sebagai ajang pembelajaran bersama, bukan sekedar penghakiman semata pada pimpinan yang akan menyampaikan laporan bertanggung jawaban. Serta yang terpilih formatur menjadi penerus yang baik dari pimpinan sebelumnya.

    Semoga karya ini bisa dilanjutkan oleh Immawan/Immawati semuanya. Fastabiqul khairat.
