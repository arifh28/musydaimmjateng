Iftitah

IMM Setengah Abad sebagai bagian dari IMM dan menisbahkan diri untuk dunia desain dan digital kreatif. Kami akan membagikan sebuah web aplikasi untuk kegiatan permusyaratan, dalam hal ini adalah Musyawarah Daerah.

Seperti yang kami sebutkan diatas untuk kegiatan permusyaratan Musyawarah Daerah. Kenapa harus Musyda bukan Musycab? Berikut alasannya:

  • Musyawarah Daerah yang pesertanya satu provinsi dengan jumlah lebih dari 100 orang dengan banyak Pimpinan Cabang.
  • Dengan teritorial DPD yang luas, kader bisa langsung mengetahui mengenai kegiatan bersamanya.

Musyawarah Cabang bisa juga dibuatkan website yang seperti ini, karena pimpinan cabang masih mempunyai delegasi dari komisariatnya seperti layaknya Musyawarah Daerah dengan delegasi dari Pimpinan Cabang.

Keuntungan

Immawan/Immawati, dengan menggunakan website sebagai basis informasi keuntungannya antara lain:

  • Panitia bisa langsung menghitung jumlah peserta dengan pasti sehingga penempatan ruang istirahat bisa diketahui.
  • Semua kader bisa menjangkaunya, karena berbasiskan website.
  • Panitia Pemilihan bisa menghitung jumlah calon formatur yang masuk setiap Pimpinan Cabang langsung dari Website.
  • Pimpinan Cabang dengan mudah mengupload berkas calon formatur dan daftar peserta Musyda.
  • Dilengkapi dengan Admin panel untuk manajemen user Pimpinan Cabang.

Fitur

Fitur dalam website aplikasi permusyawaratan ini antara lain:

1. User Pimpinan Cabang terdiri dari :

  • Daftar Peserta Musyda
  • Daftar Calon Formatur
  • Download
  • Profil Cabang
  • Profil User

2. User Panitia terdiri dari :

  • Melihat daftar Peserta
  • Data Cabang
  • Profil Panitia
  • Ganti Password

3. User Panlih terdiri dari :

  • Melihat Daftar Calon Formatur
  • Menerima atau menolak Calon Formatur
  • Mendownload berkas Calon Formatur

4. User Admin terdiri dari :

  • Menambah Operator Pimpinan Cabang dan Admin
  • Mengatur tampilan website
  • Memoderasi Pesan Musyda

Lisensi

Web aplikasi ini lisensinya MIT. Admin tidak membuat dari nol aplikasi website ini, namun modifikasi dari sourcecode yang dibagikan gedelumbung.com untuk keperluan IMM. Admin sudah meminta izin untuk bisa disebarkan secara gratis.

Installasi

Dalam installasi website ini seperti website pada umumnya yaitu letakkan di folder website. Jika Immawan/Immawati sudah punya webserver sendiri di internet, silahkan upload menggunakan aplikasi ftp kesayangan Immawan/Immawati. Tapi jika masih di webserver lokal dengan menggunakan Apache, PhpMyAdmin (XAMPP/LAMP/WAMP)langsung ditaruh di folder tersebut.

Karena ini menggunakan framework PHP Codeigniter, bagi Immawan/Immawati yang belum terbiasa menggunakan silahkan ikuti langkahnya dibawah ini.

  • Konfigurasi Database

Database web aplikasi ini sudah disediakan dalam bentuk .sql dengan nama musyda.sql yang ada di GitHub Admin. Untuk konfigurasi sendiri Immawan/Immawati bisa lihat pada file database.php dalam folder application/config/database.php seperti ini.

      $db['default']['hostname'] = 'localhost';

      $db['default']['username'] = 'root';

      $db['default']['password'] = '';

      $db['default']['database'] = 'musyda';

      $db['default']['dbdriver'] = 'mysql';

Admin mencontohkan pada server lokal, jika Immawan/Immawati menyewa hosting sendiri hubungi penyedia hosting untuk ['hostname'] namun biasanya tetap 'localhost', untuk username dan password ada pada CPanel dengan membuatnya secara manual.

  • Menentukan Base Site URL

Framework CodeIgniter untuk menentukan konfigurasi Base URL ada pada application/config/config.php. Base URL ini berfungsi untuk menentukan alamat utama dari website Immawan/Immawati.

          $config['base_url']	= 'http://'.$_SERVER['SERVER_NAME'].'/musydaimmjateng/';

Pada contoh menggunakan .$_SERVER['SERVER_NAME']. karena pada contoh web aplikasi ini bisa dikunjungi oleh komputer lain pada server lokal yang sama lewat jaringan LAN. Namun jika Immawan/Immawati menyewa hosting, baris ini diisi dengan nama domain/ subdomain website ini. Atau dimana Immawan/Immawati menaruh foldernya contohnya http://tanfidz.dppimm.or.id/musydaimm.
